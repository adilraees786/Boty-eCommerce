# 🛒 Boty eCommerce

**Boty eCommerce** is a modern, high-performance shopping platform built with the latest web technologies. Designed for speed, aesthetics, and a seamless user experience, it serves as a robust foundation for next-generation online stores.

---

## 🚀 Key Features

- **✨ Cutting-edge Tech Stack:** Built with React 19 and Next.js 16 (App Router).
- **🎨 Premium UI/UX:** Styled with **Tailwind CSS 4** for lightning-fast performance and modern aesthetics.
- **🛍️ Advanced Shopping Cart:** Fully functional cart system using React Context API and drawer-based navigation.
- **📱 Fully Responsive:** Optimized for mobile, tablet, and desktop screens.
- **⚡ Performance Optimized:** Server-side rendering and optimized image handling for quick load times.
- **🧩 Modular Architecture:** Highly reusable components following best practices.
- **🎯 SEO & Accessibility:** Semantic HTML and meta-tags for better search engine visibility.

---

## 🛠️ Technology Stack

| Feature        | Technology                                                                |
| -------------- | ------------------------------------------------------------------------- |
| **Framework**  | [Next.js](https://nextjs.org/) (App Router)                               |
| **UI Library** | [React 19](https://reactjs.org/)                                          |
| **Styling**    | [Tailwind CSS 4](https://tailwindcss.com/)                                |
| **Components** | [Shadcn UI](https://ui.shadcn.com/) (Radix UI)                            |
| **Icons**      | [Lucide React](https://lucide.dev/)                                       |
| **Validation** | [Zod](https://zod.dev/) & [React Hook Form](https://react-hook-form.com/) |
| **Animations** | [Tailwind Animate](https://github.com/jamiebuilds/tailwindcss-animate)    |

## 📂 Project Structure

The codebase is organized following modern Next.js conventions:

```bash
Boty-eCommerce/
├── 📁 app/               # Next.js App Router (Pages, Layouts, CSS)
│   ├── page.tsx          # Main Landing Page
│   └── layout.tsx        # Global Layout and Providers
├── 📁 components/        # Reusable UI Components
│   ├── 📁 boty/          # Domain-specific components (Header, Hero, Cart, etc.)
│   └── 📁 ui/            # Shadcn UI primitives (Buttons, Inputs, etc.)
├── 📁 hooks/             # Custom React Hooks
├── 📁 lib/               # Utility functions and shared library code
├── 📁 public/            # Static assets (Images, Fonts, Icons)
├── 📁 styles/            # Global CSS and Tailwind configurations
└── package.json          # Project dependencies and scripts
```

## 🚦 Getting Started

### 1. Prerequisites

- **Node.js**: 18.x or higher
- **npm** or **yarn**

### 2. Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/adilraees786/Boty-eCommerce.git
cd Boty-eCommerce
npm install
```

### 3. Running Locally

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
